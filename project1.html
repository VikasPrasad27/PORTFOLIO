    PROJECT OVERVIEW

    I developed a responsive sports website, One Football, which provides real-time highlights, news, and updates from top football leagues around the world. The site integrates various APIs to display recent match highlights, live scores and Sports news. Built using React.js, the website ensures smooth navigation and dynamic content rendering, offering users an engaging experience on all devices.

    The project utilizes Fetch and Axios for efficient API calls, enabling fast data retrieval for live updates. With its responsive design and seamless performance across mobile, tablet, and desktop, One Football delivers an intuitive and interactive platform for football fans to stay updated with the latest in the sport.
